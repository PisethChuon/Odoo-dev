# Project Hostel

An Odoo 17 module for comprehensive Hotel Management.

## Features

*   **Hostel Management Core:** Provides the fundamental models and workflows to manage hostel operations.
*   **Goods & Services Management:** Introduces a new data model to efficiently create and manage additional goods and services offered by the hostel (e.g., laundry, tours, equipment rental).
*   **Integrated Sales:** Seamlessly integrates with the Odoo Sales module to handle invoicing for bookings and services.

## Requirements

*   Odoo Community Edition 17.0
*   Dependencies:
    *   `base`
    *   `sale`

## Installation

1.  Clone this repository or add the module to your Odoo addons path.
2.  Update the Apps list in Odoo.
3.  Search for "Project Hostel" in the Apps menu and install the module.

## Usage

After installation, the module will add new menus and functionalities related to hostel management. You can start by configuring the available goods and services, which can then be linked to customer bookings and sales orders.
